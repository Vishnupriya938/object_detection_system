# object_detection_system
An object detection system is a computer vision method that identifies and locates objects in images or videos. It detects what the objects are and marks their positions using bounding boxes. It is used in areas like security, autonomous driving, and surveillance.
## Object Detection using YOLOv3

This project uses YOLOv3 for object detection.

### Download Weights File
The YOLOv3 weights file is not included due to size limits.

Download it from here:
https://pjreddie.com/media/files/yolov3.weights

### Setup
1. Download the weights file from the above link
2. Place `yolov3.weights` in the project folder
3. Run the code
